Cleaning Process

The following steps were applied to clean the dataset:

Handling Missing Values:

Identified null or empty cells.

Dropped rows/columns only when essential or irrelevant.

Data Type Conversion:

Converted datetime columns to date where appropriate.

Ensured all numeric columns are properly typed for analysis.

Duplicate Removal:

Checked for and removed duplicate rows to ensure unique records.

Standardization:

Standardized column names, e.g., lowercase, underscores instead of spaces.

Standardized categorical values where needed (e.g., Male/Female).

Validation Checks:

Verified ranges for numeric columns.

Ensured all categorical columns contain expected values.
